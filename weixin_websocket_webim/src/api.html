WebService服务地址：http://oa.atwasoft.net:7001/portal/services/IMService?wsdl
方法 说明
getVer()返回IM服务的版本号
getDura()获取IM服务器运行时长
sendIM(long to, String msg)发送系统信息，接受人收到来自小秘书的信息
sendUserIM(String to, String msg)通过用户名发送信息
sendIMtoAll(String msg)给所有在线用户发送消息
isOnline(long userid)检查用户是否在线
isUserOnline(String username)检查用户是否在线
getAllOnlineUsers()获取所有在线用户
getAllGroups(long userid)获取用户的所在工作组
pushCall(long userid, String methodName, String param)向用户前端推送脚本执行，param是对象形式的字符串，如{name:'xxx'}
pushCall1(String username, String methodName, String param)向用户前端推送脚本执行
sendSms(long userid, String msg)发送短信
sendMail(long userid, String subject, String body)发送电子邮件
......
前端JavaScript调用API
方法 说明
IM.openMainWin()打开chat主窗口,开关方法
IM.openUser(userid)打开指定用户的对话框
IM.regMethod(funName, fun)注册方法，用于服务器推送信息
IM.checkMethod()检查所有注册方法
IM.notify(title, body)发送消息提醒，如果支持浏览器的notify，就直接使用，否则右下角弹出消息框
