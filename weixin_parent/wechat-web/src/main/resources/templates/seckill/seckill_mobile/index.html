未划线价格：商品的实时标价，不因表述的差异改变性质。具体成交价格最终以订单结算页价格为准。

折扣:如无特殊说明,折扣指销售商在原价或划线价(如品牌专柜标价、商品吊牌价、厂商指导价、厂商建议委售价)等某一价格基础上计算出的优惠比例或优惠金额;如有疑问,您可在购买前联系销售商进行咨询。

异常问题:商品促销信息以商品详情页的信息为准;商品的具体售价以订单结算页价格为准;如您发现活动商品售价或促销信息有异常,建议购买前先联系销售商咨询。
